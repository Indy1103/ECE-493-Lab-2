# Quickstart: View Registration Prices

## Preconditions

- CMS is operational.
- A registration price list is published for the success scenario.

## Scenario 1: View Published Prices (UC-17-S1)

1. Access the CMS as an attendee.
2. Request the registration price list.
3. Confirm the published list renders with multiple attendance options.

## Scenario 2: Prices Unavailable (UC-17-S2)

1. Ensure no published registration price list exists.
2. Request the registration price list.
3. Confirm an explicit unavailability message is displayed and no prices are shown.
